### Product Listing Page (PLP) Structure

Based on competitor analysis:
```
1. Breadcrumbs
2. Category title + product count
3. Filter sidebar (Price, Metal, Purity, Weight, Stone, Occasion, Gender)
4. Sort options (Price Low-High, High-Low, New Arrivals, Popularity)
5. Product grid (3-4 columns desktop, 2 mobile)
6. Each card: Image, Name, Price, Weight/Purity badge, Wishlist icon, Quick View
7. Pagination / Infinite scroll
8. "Recently Viewed" section at bottom
```

### Product Detail Page (PDP) Structure

Based on competitor analysis:
```
1. Breadcrumbs
2. Product images (gallery with zoom, 3-5 images)
3. Product name + short description
4. Price (with making charges breakdown)
5. Metal: type, purity, weight
6. Stone details (if applicable): type, carat, clarity, color
7. Size selector (for rings/bangles)
8. Quantity selector
9. Add to Cart + Buy Now buttons
10. Wishlist + Share buttons
11. Delivery pincode checker
12. EMI options / Payment plans
13. Trust badges (BIS hallmark, certified, free shipping)
14. Detailed description tab
15. Specifications tab (full metadata table)
16. Reviews tab
17. "Complete the Look" - matching products
18. "Similar Products" carousel
19. "Recently Viewed" carousel
```
